About

Built as a companion to Trench Compendium, this website allows you to take a downloaded JSON file for your warband and print it out, for easy reference during paper games. We really wanted this functionality, so here it is for anyone to use. Feel free to reach out for any bug reports or feature requests. You can find us here.
